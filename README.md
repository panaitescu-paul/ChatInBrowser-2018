# ChatInBrowser-2018

### Features:
- Chat in Browser 
- Multiple clients in the same time (browser tabs)
- Timestamps for comments
- Emoji
- Randomly generated nicknames

### Technologies:
- Node JS
- JQuery
- HTML, CSS, JS
